本项目暂时包含四个实验内容，分别为：
1，用链表实现多项式加法和乘法。
2，用栈实现计算器，支持四则运算与括号(不支持中括号和大括号)。
3，用哈夫曼树实现文本的二进制加密与解密。
4，用AVL树实现可动态插入和删除数据的搜索树结构。
